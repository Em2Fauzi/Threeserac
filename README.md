# Threeserac
MATLAB-based purely numerical relativistic geodesic(s) code in static and spherically symmetric spacetime with simplified geodesic equation on the equatorial plane. Can be used to:

1. Analyze particle trajectories around static and spherically symmetric spacetime on the equatorial plane
2. Generate images of 'non-rotating body' surrounded by thin accretion disk. The body itself is by default considered transparent, unless there exist an event horizon.

Well tested on MATLAB 2023b. Not a well-optimized code, but it works.. so, yap. feel free to use it :)
